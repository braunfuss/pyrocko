The crustdb module¶

The Global Crustal Database gathers empirical 1D velocity models from seismic reflection and refraction profiles. As of 2013 there are 138939 profiles, mainly P and fewer S wave records in the database.

See the tutorial for usage example, Accessing crustal velocity databases.

Note

Please cite:

W.D. Mooney, G. Laske and G. Masters, CRUST 5.1: A global crustal model at 5°x5°. J. Geophys. Res., 103, 727-747, 1998.

class CrustDB(database_file=None, parent=None)[source]¶

CrustDB is a container for VelocityProfile and provides functions for spatial selection, querying, processing and visualising data from the Global Crustal Database.
